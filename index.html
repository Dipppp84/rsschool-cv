Code example

Your job is to write a function which increments a string, to create a new string.

  • If the string already ends with a number, the number should be incremented by 1.
  • If the string does not end with a number. the number 1 should be appended to the new

function incrementString(strng) {
    const index = strng.search('[0-9]+$');
    if (index === -1)
        return strng + 1;
    const numbers = strng.slice(index);
    const str = strng.slice(0, index);
    const zeroLength = numbers.length;
    return str + String(+numbers + 1).padStart(zeroLength, '0');
}
